## 安装步骤

0. 准备工作

- 在你的机器上安装好docker

1. 克隆仓库

```
git clone https://github.com/QUSEIT/nextjs-project-skel.git

# OR git clone git@github.com:QUSEIT/nextjs-project-skel.git
```

2. 进入对应目录

```
cd nextjs-project-skel
```

3. 构建docker镜像

- 启动docker
- 执行build开发镜像
```
make build
```

4. 启动docker

```
make up
```
